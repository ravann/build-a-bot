# build-a-bot

## Environment setup

1) Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.36.0/install.sh | bash

2) Install node
nvm install 12.16.3
nvm install 12.18.4
nvm use 12.16.3

3) Install vue cli
npm install -g @vue/cli@4.3.1

4) Create a project
vue create build-a-bot


## Project setup
```
npm install
```
